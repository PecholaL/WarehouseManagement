# WarehouseManagement
learning project of SpringBoot  

## 学习内容
后端：SpringBoot + mybatis-plus + MySQL  
前端：Node.js + VUE + Element-UI

## 功能
1. 登录/退出
2. 管理
   1. 管理员管理
   2. 用户管理
   3. 仓库管理
   4. 物品分类管理
   5. 记录管理

## 学习过程
1. 配置pom.xml，添加SpringBoot、MyBatis等依赖
2. 创建数据库、设计表格

## 版本
JDK 18  
SpringBoot 2.6.7  
MariaDB 10.4.27
